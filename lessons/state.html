物質の状態変化

物質の三態

  • 固体: 粒子が規則正しく並び、振動している
  • 液体: 粒子が自由に動き回るが、密度が高い
  • 気体: 粒子が激しく運動し、自由に飛び回る

状態変化の名前

融解
固体 → 液体
凝固
液体 → 固体
蒸発
液体 → 気体
凝縮
気体 → 液体

重要ポイント

  • • 状態変化では物質の性質は変わらない
  • • 温度によって状態が決まる
  • • 融点・沸点は物質固有の性質
  • • 質量は変化しない
